Concept-Guided Fine-Tuning:
Steering ViTs away from Spurious Correlations

Yehonatan Elisha1, Oren Barkan2, Noam Koenigstein1
1Tel Aviv University, 2The Open University
Paper arXiv Code

The Problem: Standard ViTs often fixate on backgrounds (middle). The Solution: CFT steers the model to focus on fine-grained semantic concepts (e.g., beaks, wings, fins), leading to massive gains in out-of-distribution robustness.

Abstract

Vision Transformers (ViTs) often degrade under distribution shifts because they rely on spurious correlations, such as background cues, rather than semantically meaningful features. Existing regularization methods fail to capture the fine-grained semantic concepts that define an object.

We introduce Concept-Guided Fine-Tuning (CFT), a framework that aligns model reasoning with spatially grounded concept masks. Our method automatically proposes class-relevant concepts using an LLM and localizes them using a VLM. We demonstrate that CFT improves robustness across five OOD benchmarks while maintaining in-distribution accuracy.

Key Contributions

Beyond Foreground Masks

Moves past binary masks to hierarchical concepts (e.g., "long beak", "fins"), providing superior supervision for robustness.

Fully Automated

Zero manual annotation. Concepts are proposed by an LLM and spatially grounded by a VLM (GroundedSAM).

Data Efficient

Requires only a minimal set of images (3 per class) to achieve state-of-the-art robustness gains.

How CFT Works

  1. Concept Discovery: An LLM (GPT-4o-mini) generates class-specific, discriminative attributes (e.g., "curved beak" for a parrot).
  2. Spatial Grounding: GroundedSAM localizes these concepts within training images, creating an adaptive guidance mask.
  3. Optimized Alignment: We use AttnLRP to compute internal relevance maps and optimize the model to align with concept regions while suppressing spurious background noise.
  4. Consistency: A classification-consistency loss ensures the model remains a high-performance classifier.

Empirical Robustness

CFT consistently outperforms state-of-the-art baselines across real-world and synthetic shifts.

Benchmark Standard ViT-B CFT (Ours) Improvement
ImageNet-A (Adversarial) 13.26% 27.76% +109%
ObjectNet (Pose/Viewpoint) 33.26% 54.28% +63%
ImageNet-R (Artistic) 30.26% 48.47% +60%

Correcting Model Failures

Case Study: On ImageNet-A, the baseline model misclassifies a "common newt" as a "scorpion" because it attends to the textured background. CFT corrects this by forcing the model to focus on the animal's body and semantic features, leading to the correct prediction.
